## What is Blob Storage?

Object storage system designed to store unstructured data (images, videos, files) at massive scale.

```
Object = File + Metadata + Unique Key
```

## Interview Questions

**Why not store images directly in a database?**
Binary files bloat the database, slow down backups/queries, and databases aren't optimized for large sequential byte storage.

**What is the difference between a bucket and an object?**
A bucket is the container/namespace; an object is the actual file stored inside it along with its key and metadata.

**How does CDN help with blob storage?**
It caches objects at edge locations closer to users, reducing latency compared to fetching directly from the origin storage.

---

## Summary

- Blob storage is designed for large unstructured files.
- Database stores only a reference URL, not the file itself.
- S3 offers multiple storage classes based on access frequency.
- CDN integration speeds up global file delivery.
